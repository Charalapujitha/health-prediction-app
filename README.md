# AI-Based Patient Health Risk Assessment System

## Overview

This project is a Health Prediction Application developed using Python, Streamlit, and SQLite.

The application allows users to:

* Add patient records
* View patient records
* Update patient records
* Delete patient records
* Validate patient information
* Generate health risk predictions based on blood test parameters

## Technologies Used

* Python
* Streamlit
* SQLite
* Pandas

## Features

* CRUD Operations
* Data Validation
* Persistent Database Storage
* Health Risk Assessment
* User-Friendly Interface

## How to Run

1. Install required packages

pip install -r requirements.txt

2. Run the application

streamlit run app.py
